# /rp — Reasoning Personas Shortcut

Thin public-safe alias for `reasoning-personas`.

## Behavior

When `/rp` is used, run a compact multi-persona analysis:

1. Pattern Hunter
2. Gonzo Truth-Seeker
3. Devil's Advocate
4. Integrator
5. Final recommendation in 1-3 bullets

Do not use this for simple deterministic tasks. It is for decisions, risks, architecture, plans, and tradeoffs.
